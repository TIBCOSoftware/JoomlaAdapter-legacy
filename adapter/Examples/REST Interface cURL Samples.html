APIS

Get APIS by ID

Get APIS

curl -X GET -H "Cache-Control: no-cache" -H 'http://ggrigore-z800.na.tibco.com/dev/v1/apis'

Put APIS

curl -X PUT -H "Cache-Control: no-cache" -H -H "Content-Type: multipart/form-data; " -F "size=original" -F "id={{id}}" -F "title={{title}}" -F "description={{description}}" -F "type={{type}}" -F "environment={{environment}}" -F "targetEnvironment={{targetEnvironment}}" -F "file={{file}}" 'http://ggrigore-z800.na.tibco.com/dev/v1/apis'

Post APIS

curl -X POST -H "Cache-Control: no-cache" -H "Content-Type: multipart/form-data; " -F "size=original" -F "title={{title}}" -F "description={{description}}" -F "type={{type}}" -F "environment={{environment}}" -F "targetenvironment={{targetenvironment}}" -F "file={{file}}" 'http://ggrigore-z800.na.tibco.com/dev/v1/apis'

Delete APIS

curl -X DELETE -H "id: {{id}}" -H "Cache-Control: no-cache" -H "Content-Type: multipart/form-data; " -F "size=original" 'http://ggrigore-z800.na.tibco.com/dev/v1/apis'
